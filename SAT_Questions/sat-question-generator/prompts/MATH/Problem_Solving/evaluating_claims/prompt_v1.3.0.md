Generate an SAT Math question about evaluating statistical claims.

The question should present:
- A study or survey scenario with clear sample information
- Claims or conclusions drawn from the data
- Need to evaluate validity of ONE specific conclusion

CRITICAL REQUIREMENTS:

Statistical Rigor:
- Clearly distinguish between sample descriptions and population inferences
- If testing inference validity, include relevant sampling methodology details (random selection, sample size, population defined)
- The correct answer must be defensible using ONLY concepts covered in high school math: random sampling, sample size considerations, bias sources, causation vs correlation
- NEVER reference or require knowledge of: p-values, confidence intervals, margins of error, hypothesis testing, or "statistical significance"
- Avoid ambiguous phrasing like "evidence that" or "suggests that" - be explicit about whether the claim is about the sample itself or an inference to the population
- The correct answer must be the ONLY defensible option; ensure no technically correct alternatives exist

ANSWER VALIDITY — VERIFICATION REQUIRED:
Before finalizing, verify the correct answer by asking:
1. Is this the ONLY answer that a statistics-literate student could defend?
2. Could a student argue that another answer is also technically correct? If yes, redesign.
3. Does the scenario contain all necessary details to definitively support the correct answer?
4. At difficulty 4-5, is the correct reasoning path specific enough that students cannot arrive at it through vague reasoning?

FAITHFULNESS TO SAT CONVENTIONS:
- Ensure the question stem, scenario, and answer choices are internally consistent
- If a study is described as "random," the correct answer should not question randomness
- If a specific population is named, answers should refer to that exact population
- Numbers mentioned in the stem must be used accurately in the explanation and answer rationale

Difficulty Calibration:
- Easy (1-2/5): Identify obvious sampling bias (convenience sample, voluntary response) or recognize need for random sampling. Should require minimal analysis beyond recognizing one clear flaw.
- Medium (3/5): Evaluate whether a randomly selected sample supports a straightforward generalization. Should require understanding 2-3 concepts (e.g., random sampling + adequate size).
- Hard (4-5/5): Analyze multiple validity conditions simultaneously, identify subtle confounding variables, or distinguish between experimental and observational study limitations. Should require synthesizing 3+ statistical concepts.
- Match computational difficulty to conceptual difficulty: Easy questions should involve simple percentage recognition, medium questions minimal comparison calculations, hard questions multi-step reasoning about study design
- If stated difficulty is 3+, ensure the question genuinely requires critical analysis, not just reading comprehension

Distractors:
- Each distractor must represent a specific, plausible misconception that a student might actually hold
- Create competitive wrong answers that seem correct under flawed reasoning (e.g., "1000 is a large sample" ignoring non-random selection)
- Include at least 2 distractors that would require careful analysis to eliminate
- Avoid: arbitrary numerical thresholds without justification, overly strong claims easily dismissed ("can NEVER conclude"), technically correct statements
- Target proven misconceptions: equating large sample size with validity regardless of selection method, assuming correlation implies causation, ignoring volunteer bias, believing random sampling isn't necessary for large samples

Clarity and Scope:
- Keep stimulus under 100 words when possible
- Use straightforward language; avoid all technical jargon
- Test ONE clear construct: sampling validity, generalization appropriateness, OR experimental design principles - never mix multiple constructs
- Make the evaluation criterion completely explicit and unambiguous (e.g., "which conclusion is valid about ALL high school students in the state" vs "which accurately describes ONLY this sample")
- When using given percentages/statistics, ensure the question requires analytical reasoning beyond simply reading the number
- For difficulty 3+, the student must evaluate methodology or study design, not just compare given numbers
