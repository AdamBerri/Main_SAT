Generate an SAT Writing question that tests the ability to combine ideas effectively.

Provide:
- Notes or bullet points with related information (2-4 related but distinct points)
- A clear context for why this synthesis is needed (specific type of essay, report, or writing task)

The question should:
- Ask which option "most effectively" combines the given information
- Test understanding of emphasis, flow, and logical connection between ideas
- Present answer choices that vary in their effectiveness at synthesizing the information
- Include distractors that may be grammatically correct but fail to establish clear relationships between ideas, create awkward emphasis, or result in unclear logical flow

**Critical Requirements:**

**Avoid Explicit Signaling:** Do NOT use phrases that directly tell students what to emphasize or look for (e.g., "emphasize the paradox," "highlight the transformation," "stress the contrast"). The rhetorical goal should be implicit in the context, requiring students to infer what matters most.

**Ensure Source Material Clarity:** The notes must establish ONE clearly supported logical relationship or causal mechanism. Avoid notes that could justify multiple competing interpretations of how ideas connect. Test whether your notes support exactly one "most effective" synthesis before finalizing.

**Create Genuinely Distinct Distractors:** Each distractor must represent a UNIQUE failure mode:
- Different content priorities (not just rearrangements)
- Distinct logical flaws (missing causation vs. reversed relationship vs. incomplete synthesis vs. wrong emphasis)
- Non-overlapping weaknesses (if one distractor "misses why X matters," others should fail differently—perhaps by overstating, inverting logic, or omitting context)
- Test: Can you articulate a separate, specific student misconception for each distractor?

**Validate Answer Defensibility:** Before finalizing, actively try to argue FOR each distractor:
- Can a sophisticated reader justify a distractor using the source notes? If yes, revise.
- Does the correct answer use concepts/phrasing not present in strong distractors? If this is the ONLY distinction, revise.
- Would removing one piece of information from the notes make a distractor equally valid? If yes, add clarifying detail to notes.

**Calibrate Difficulty Accurately:**
- Difficulty 1-2: Correct answer clearly superior; distractors have noticeable gaps in logic or completeness; task goal relatively transparent
- Difficulty 3: Requires comparing 2-3 plausible options with subtle distinctions; task goal requires some inference
- Difficulty 4-5: All options appear initially reasonable; distinctions rest on careful evaluation of logical relationships, emphasis appropriateness, or synthesis completeness; task goal significantly implicit. AVOID binary structures that enable simple elimination at this level.

**For Difficulty 4-5 Specifically:**
- Ensure the task context is complex enough that "most effective" is not immediately obvious
- Avoid question frames where one criterion (like causation) definitively eliminates 3 choices
- Create notes where multiple logical connections exist, but one is primary/best-supported

**Self-Check Before Finalizing:**
1. Do my notes support exactly one logical interpretation as strongest?
2. Does each distractor fail in a unique, specific way?
3. Could I defend the correct answer over distractors to a skeptical reader using only the source notes?
4. Is my difficulty rating accurate given task transparency and answer decisiveness?
