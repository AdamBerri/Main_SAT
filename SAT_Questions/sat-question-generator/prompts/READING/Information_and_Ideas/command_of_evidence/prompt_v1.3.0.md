Generate an SAT Reading question that tests the student's ability to identify textual evidence supporting a claim.

The passage should:
- Present a clear central claim or interpretation in the introductory context
- Contain 4-5 distinct pieces of evidence (quotes, data, examples) that vary in their relevance to the claim
- Use authentic academic prose with clear attribution for any quotes
- Avoid meta-analytical commentary that wouldn't appear in source material (e.g., "Her work catalyzed..." or "This demonstrates...")
- Keep direct quotes clearly marked and distinct from paraphrased content

The question should:
- Ask which quote/evidence "best supports" or "most directly supports" a specific, precisely-worded claim
- Ensure the correct answer provides the MOST DIRECT and SPECIFIC support through:
  * PRIMARY SOURCE data/quotes that directly illustrate the claim (strongly prefer these)
  * The FEWEST inferential steps required
  * The MOST COMPLETE coverage of the claim's core elements
  * Clear cause-effect or claim-evidence alignment
- Have distractors that are:
  * Actually from the passage
  * Topically relevant and substantive enough to require careful evaluation
  * Each supporting a related but DISTINCT aspect (e.g., different stage of process, different mechanism, adjacent concept)
  * Comparable in specificity level to the correct answer (avoid mixing concrete data with vague statements)
  * Not defensible as equally valid—if uncertain, apply this test: "Does this option demonstrate the COMPLETE claim with FEWER inferential leaps than alternatives?"

ANSWER VALIDATION PROTOCOL (complete BEFORE finalizing):
1. List what the claim specifically asserts (all key elements)
2. For each answer choice, count inferential steps needed and note which claim elements it addresses
3. If the designated correct answer requires MORE inference than another option, OR addresses FEWER claim elements, designate the more direct option as correct
4. Ensure at least 2 distractors are substantive enough that a competent reader would need 10+ seconds to eliminate them

Difficulty calibration requirements:
- EASY (2/5): Correct answer uses near-identical language to the claim; minimal inference needed; distractors obviously off-topic or lacking key claim elements
- MEDIUM (3/5): Correct answer requires recognizing paraphrase OR connecting one logical step; distractors topically relevant but each clearly addresses different sub-claims when carefully compared
- HARD (4-5/5): Correct answer requires synthesizing implicit evidence OR distinguishing between multiple options that ALL seem to address the claim on first read; differences lie in directness/completeness

DIFFICULTY RATING PROTOCOL (complete AFTER answer validation):
1. Check for vocabulary overlap: If correct answer echoes 3+ key words from claim → maximum 2/5
2. Check for explicitness: If correct answer explicitly states the claim's mechanism/relationship without requiring inference → maximum 3/5
3. Check distractor strength: Count how many distractors a strong reader would seriously consider (need 5+ seconds to eliminate)
   - 0-1 strong distractors → maximum 3/5
   - 2+ strong distractors → eligible for 4-5/5
4. Only rate 4-5/5 if ALL these conditions are met:
   - Answer requires synthesis OR comparing directness across multiple plausible options
   - At least 2 distractors address overlapping claim elements (not entirely separate topics)
   - No vocabulary overlap between question and correct answer
5. COMMON RATING ERROR: Topics that sound sophisticated do not make questions difficult. Rate based on actual solving process.

Final validation checklist:
1. Applied answer validation protocol and confirmed correct answer requires fewest inferential steps
2. Applied difficulty rating protocol and rating reflects ACTUAL cognitive demand (not topic complexity)
3. At least 2 distractors are substantive and topically proximate (not easily eliminated due to wrong timeframe, missing key actors, or addressing obviously different concepts)
4. All quoted material could plausibly appear in an authentic source passage
